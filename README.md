# Res-LAB-Construindo-Arquiteturas-Azure

🌐 AZ-900 – Construindo Arquiteturas no Azure
🧪 Resumo do Laboratório
Este repositório documenta a prática realizada durante o laboratório de fundamentos da certificação AZ-900, com foco em conceitos essenciais de arquitetura e navegação no Microsoft Azure.

🚀 Conteúdos Explorados
🔭 1. Datacenter Azure Global
Exploração do mapa global interativo do Azure, incluindo:

Pares de regiões (Region Pairs);
Zonas de disponibilidade (Availability Zones);
Visualização amigável e detalhada da distribuição geográfica dos datacenters.


🏢 2. “The Lobby” – Microsoft
Utilização do ambiente imersivo The Lobby, com foco em:

Navegação e usabilidade da plataforma;
Entendimento inicial dos principais serviços e categorias disponíveis no Azure.


📦 3. Criação de Grupo de Recursos

Criação de um Resource Group para organização e gerenciamento de recursos do laboratório.


🔐 4. IAM – Controle de Acesso
Configuração do Access Control (IAM) no grupo de recursos:

Atribuição de funções via RBAC;
Entendimento de permissões de acesso e escopos.


🛡️ 5. Bloqueios, Permissões e Marcações

Aplicação de bloqueios (ReadOnly / Delete);
Utilização de tags para classificação;
Revisão de permissões herdadas.


📋 6. Exploração do Menu Principal do Azure
Navegação pelas categorias do portal, com foco no entendimento:

Compute
Networking
Storage
Segurança
Identidade
E muito mais…

O objetivo foi compreender as opções, sem criação adicional de recursos.

🌐 7. Criação de Rede Virtual

Criação de uma Virtual Network (VNet), explorando:

Espaço de endereçamento;
Sub-redes;
Relação da VNet com a arquitetura geral do Azure.
